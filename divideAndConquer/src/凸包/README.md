第一步：把给定点集中的点在横坐标方向上按照大小排序。

 ，如下图所示。显然直线段Pmax与直线段S1把点集
分成了三个集合。由凸包的性质可知       三点围成的三角形中的点不可能作为凸包的顶点，所以只需考虑直线    左边的点 以及直线左边的点   。
