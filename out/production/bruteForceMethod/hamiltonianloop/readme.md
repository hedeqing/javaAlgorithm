使用蛮力法寻找哈密顿回路的基本思想是：对于给定的无向图G=(V, E)，首先生成图中所有顶点的排列对象(vi1, vi2, …, vin)，然后依次考察每个排列对象是否满足以下两个条件：
	
	（1）相邻顶点之间存在边，即
	              (vij, vij+1)∈E（1≤j≤n-1）
	（2）最后一个顶点和第一个顶点之间存在边，即
	               (vin, vi1)∈E
	    满足这两个条件的回路就是哈密顿回路。
